# [面试题 16.06. 最小差][cnTitle]

**中等**

给定两个整数数组 *a* 和 *b* ，计算具有最小差绝对值的一对数值（每个数组中取一个值），并返回该对数值的差



**示例：** 

```
输入：{1, 3, 15, 11, 2}, {23, 127, 235, 19, 8}
输出：3，即数值对(11, 8)

```



**提示：** 

-  *1 <= a.length, b.length <= 100000*  
-  *-2147483648 <= a[i], b[i] <= 2147483647*  
- 正确结果在区间  *[0, 2147483647]*  内
