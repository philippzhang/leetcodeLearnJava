# [50. Pow(x, n)][enTitle]

**Medium**

 *x* 
 *n* 

**Example 1:** 

```
Input: 2.00000, 10
Output: 1024.00000

```

**Example 2:** 

```
Input: 2.10000, 3
Output: 9.26100

```

**Example 3:** 


<pre><strong>Input:</strong> 2.00000, -2
<strong>Output:</strong> 0.25000
<strong>Explanation:</strong> 2<sup>-2</sup> = 1/2<sup>2</sup> = 1/4 = 0.25
</pre>

**Note:** 

- -100.0 <  *x*  < 100.0 
-  *n*  is a 32-bit signed integer, within the range [−2<sup>31</sup>, 2<sup>31 </sup>− 1]
