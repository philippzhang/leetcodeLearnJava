# [78. Subsets][enTitle]

**Medium**

Given a set of **distinct**  integers,  *nums* , return all possible subsets (the power set).

**Note:**  The solution set must not contain duplicate subsets.

**Example:** 

```
Input: nums = [1,2,3]
Output:
[
  [3],
  [1],
  [2],
  [1,2,3],
  [1,3],
  [2,3],
  [1,2],
  []
]
```
# [78. 子集][cnTitle]

**中等**

给定一组**不含重复元素** 的整数数组  *nums* ，返回该数组所有可能的子集（幂集）。

**说明：** 解集不能包含重复的子集。

**示例:** 

```
输入: nums = [1,2,3]
输出:
[
  [3],
  [1],
  [2],
  [1,2,3],
  [1,3],
  [2,3],
  [1,2],
  []
]
```
