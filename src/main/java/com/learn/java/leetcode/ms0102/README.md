**示例 1：** 

```
输入: s1 = "abc", s2 = "bca"
输出: true 

```

**示例 2：** 

```
输入: s1 = "abc", s2 = "bad"
输出: false

```

**说明：** 

-  *0 <= len(s1) <= 100*  
-  *0 <= len(s2) <= 100*
