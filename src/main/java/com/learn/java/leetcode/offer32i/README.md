# [剑指 Offer 32 - I. 从上到下打印二叉树][cnTitle]

**中等**

从上到下打印出二叉树的每个节点，同一层的节点按照从左到右的顺序打印。



例如: 给定二叉树:  *[3,9,20,null,null,15,7]* ,

```
    3
   / \
  9  20
    /  \
   15   7

```

返回：

```
[3,9,20,15,7]

```



**提示：** 

1.  *节点总数 <= 1000*
