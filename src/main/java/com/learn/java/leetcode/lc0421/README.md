# [421. Maximum XOR of Two Numbers in an Array][enTitle]

**Medium**

Given an integer array  *nums* , return  *the maximum result of nums[i] XOR nums[j]* , where  *0 ≤ i ≤ j < n* .

**Follow up:**  Could you do this in  *O(n)*  runtime?



**Example 1:** 

```
Input: nums = [3,10,5,25,2,8]
Output: 28
Explanation: The maximum result is 5 XOR 25 = 28.
```

**Example 2:** 

```
Input: nums = [0]
Output: 0

```

**Example 3:** 

```
Input: nums = [2,4]
Output: 6

```

**Example 4:** 

```
Input: nums = [8,10,2]
Output: 10

```

**Example 5:** 

```
Input: nums = [14,70,53,83,49,91,36,80,92,51,66,70]
Output: 127

```



**Constraints:** 

-  *1 <= nums.length <= 2 * 104*  
-  *0 <= nums[i] <= 231 - 1* 


# [421. 数组中两个数的最大异或值][cnTitle]

**中等**

给定一个非空数组，数组中元素为 a<sub>0</sub>, a<sub>1</sub>, a<sub>2</sub>, … , a<sub>n-1</sub>，其中 0 ≤ a<sub>i</sub> < 2<sup>31 </sup>。

找到 a<sub>i</sub> 和a<sub>j </sub>最大的异或 (XOR) 运算结果，其中0 ≤  *i* ,  *j*  <  *n* 。

你能在O( *n* )的时间解决这个问题吗？

**示例:** 

```
输入: [3, 10, 5, 25, 2, 8]

输出: 28

解释: 最大的结果是 5 ^ 25 = 28.

```
