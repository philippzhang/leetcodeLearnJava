# [59. Spiral Matrix II][enTitle]

**Medium**

Given a positive integer  *n* , generate a square matrix filled with elements from 1 to  *n* <sup>2</sup> in spiral order.

**Example:** 

```
Input: 3
Output:
[
 [ 1, 2, 3 ],
 [ 8, 9, 4 ],
 [ 7, 6, 5 ]
]

```
