# [面试题 16.16. 部分排序][cnTitle]

**中等**

给定一个整数数组，编写一个函数，找出索引 *m* 和 *n* ，只要将索引区间 *[m,n]* 的元素排好序，整个数组就是有序的。注意： *n-m* 尽量最小，也就是说，找出符合条件的最短序列。函数返回值为 *[m,n]* ，若不存在这样的 *m* 和 *n* （例如整个数组是有序的），请返回 *[-1,-1]* 。

**示例：** 

```
输入： [1,2,4,7,10,11,7,12,6,7,16,18,19]
输出： [3,9]

```

**提示：** 

-  *0 <= len(array) <= 1000000*
