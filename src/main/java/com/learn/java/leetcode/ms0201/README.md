# [面试题 02.01. 移除重复节点][cnTitle]

**简单**

编写代码，移除未排序链表中的重复节点。保留最开始出现的节点。

**示例1:** 

```
输入：[1, 2, 3, 3, 2, 1]
输出：[1, 2, 3]

```

**示例2:** 

```
输入：[1, 1, 1, 1, 2]
输出：[1, 2]

```

**提示：** 

1. 链表长度在[0, 20000]范围内。 
2. 链表元素在[0, 20000]范围内。
