# [372. Super Pow][enTitle]

**Medium**

Your task is to calculate a<sup>b</sup> mod 1337 where a is a positive integer and b is an extremely large positive integer given in the form of an array.

**Example 1:** 


```
Input:a = 2, b = [3]
Output: 8
```


**Example 2:** 

```
Input:a = 2, b = [1,0]
Output: 1024
```
